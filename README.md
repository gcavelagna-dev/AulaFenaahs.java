# AVISO IMPORTANTE

Para executar este projeto corretamente, alguns requisitos que são necessários:

Pré-requisitos
Java JDK instalado (versão 17 ou superior recomendada)
VS Code ou outra IDE Java
Extensões Java instaladas no VS Code
Como executar
Faça o download ou clone este repositório.
Abra o projeto em sua IDE de preferência.
Localize o arquivo principal do projeto.
Execute a classe principal contendo o método main.
O quiz será iniciado diretamente pelo terminal.
Observações
Todas as respostas devem ser digitadas utilizando os números das alternativas apresentadas.
Caso seja digitada uma opção inexistente, o sistema solicitará uma nova entrada.
O projeto foi desenvolvido com foco educacional e prática dos fundamentos da linguagem Java.
Sobre o código

Você encontrará algumas repetições ao longo do projeto. Isso foi proposital durante o processo de aprendizagem.

O objetivo principal não foi criar a solução mais otimizada possível, mas sim reforçar conceitos fundamentais como:

Métodos
Estruturas condicionais (if e switch)
Estruturas de repetição (while)
Variáveis
Entrada de dados com Scanner
Organização de código

Conforme meu conhecimento evoluir, pretendo refatorar partes do projeto para torná-lo mais reutilizável e escalável.

# Manual de Utilização

Execute o programa.
Leia atentamente cada pergunta apresentada no terminal.
Digite o número correspondente à alternativa desejada.
Aguarde a análise da resposta.
Em caso de erro, leia a explicação apresentada para reforçar o aprendizado.
Continue respondendo até o final do quiz.
Ao término, verifique sua quantidade de acertos.

# Notas sobre o projeto

Este projeto surgiu como uma forma de praticar Java enquanto desenvolvo algo útil para outros alunos que estão começando seus estudos em programação.

A ideia é criar um pequeno quiz executado pelo terminal, abordando conceitos básicos de HTML, CSS, JavaScript e Java. O foco não é apenas acertar perguntas, mas também aprender com os erros através das explicações apresentadas após cada resposta.

Durante o desenvolvimento, existem várias repetições de código. Isso foi feito de forma consciente neste momento. Como ainda estou consolidando conceitos fundamentais da linguagem, repetir estruturas como métodos, condicionais, laços de repetição e comandos de entrada me ajuda a compreender melhor como tudo funciona.

Sei que existem formas mais avançadas e organizadas de construir esse projeto, utilizando menos código repetido e aproveitando melhor a reutilização. Porém, neste estágio do aprendizado, estou priorizando o entendimento e a prática constante dos fundamentos.

Além do conteúdo sobre programação, este projeto também está me ajudando a desenvolver habilidades importantes como organização de código, criação de métodos, tratamento de entradas do usuário, lógica de programação e planejamento de projetos maiores.

Pretendo continuar expandindo o quiz com novas perguntas, melhorias visuais no terminal e um sistema de pontuação mais completo conforme meu conhecimento em Java evoluir.
